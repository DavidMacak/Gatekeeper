# Gatekeeper

Aplikace pro záznam vstupů a odchodů osob a vozidel. Bude obsahovat WPF a ASP.NET aplikaci pro vytváření těchto záznamů.



## Features

- WPF
- SQL stored procedures
- DI, Dapper


## Roadmap

- ASP.NET webová aplikace se stejnými funkcemi jako WPF
